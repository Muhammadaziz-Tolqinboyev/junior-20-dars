Fontlar va Shriftlar bilan ishlash

Veb-sahifaga ruh bag'ishlash: Shriftlarni to'g'ri tanlash va professional darajada ulash sirlari.

🔡 🎨 💻

Shrift o’zi nima?

Ta'rif

Shrift — bu matnning vizual ko'rinishi. U nafaqat o'qish uchun, balki foydalanuvchida kayfiyat va ishonch uyg'otish uchun xizmat qiladi.

  • ✅ Brendning uslubini belgilaydi
  • ✅ O'qishni osonlashtiradi yoki qiyinlashtiradi
  • ✅ Sahifa tezligiga ta'sir qiladi

Nega muhim?

"Shrift — bu matnning kiyimidir. Agar siz jiddiy bank sayti ochsangiz-u, shrift qo'lda yozilgandek bo'lsa, hech kim sizga ishonmaydi."

Font oilalari va turlari

Serif: "Ilgakli" shriftlar. Jiddiy va klassik.
Masalan: Times New Roman, Playfair
Sans-serif: "Ilgaksiz" shriftlar. Zamonaviy va toza.
Masalan: Arial, Jakarta Sans, Roboto
Monospace: Har bir harf bir xil kenglikda.
Masalan: Courier, JetBrains Mono (Kod uchun)
Cursive: Qo'lda yozilgan uslub.
Masalan: Dancing Script

Formatlar (Extensions)

WOFF / WOFF2 — Web uchun eng zo'ri! 🚀

TTF / OTF — Kompyuterlar uchun klassik.

SVG Font — Eskirgan format.

Google Fonts: Online ulash

Qanday ulanadi?

  1. fonts.google.com ga kiramiz
  2. Shriftni tanlab, kerakli qalinliklarni (Weight) belgilaymiz
  3. Tayyor kodni nusxalaymiz

💡 CDN (Content Delivery Network): Shriftni yuklab olmasdan, Google serveridan to'g'ridan-to'g'ri chaqirish.

<!-- HTML head qismida -->
<link href="..." rel="stylesheet">

/* CSS-da ishlatish */
body {
  font-family: 'Roboto', sans-serif;
}

<link> vs @import

Xususiyat <link> (HTML) @import (CSS)
Tezlik Parallel yuklanadi (Tezroq) 🚀 CSS yuklangach navbatga turadi
Tavsiya (Best Practice) ✅ Eng yaxshi usul ❌ Tavsiya etilmaydi
O'rin HTML <head> ichida CSS faylning eng yuqorisida

@font-face: Local font ulash

@font-face {
  font-family: 'MeningFontim';
  src: url('fonts/myfont.woff2');
}

.title {
  font-family: 'MeningFontim';
}

Nega local ulash kerak?

  • Internet o'chsa ham ishlaydi
  • Tashqi serverga bog'liq bo'lmaydi
  • Eksklyuziv fontlar uchun yagona yo'l

📁 Loyihangizda 'fonts' degan papka ochishni unutmang!

CSS Font xususiyatlari

font-family: Shrift nomini belgilaydi.

font-size: Shrift o'lchami (px, rem, em).

font-weight: Qalinligi (100 dan 900 gacha).

font-style: Qiyaligi (normal, italic).

p {
  font-family: 'Arial';
  font-size: 18px;
  font-weight: 600;
  font-style: italic;
}

Uyga vazifa

O'zingiz yoqtirgan mavzuda (masalan, sevimli o'yiningiz) bitta sahifa yarating va:

1. Google Fonts'dan 2 xil shrift ulang (Sarlavha va matn uchun).

3. Figma dizayndagi shriftni o'lchamlarini aniq ko'chirib o'tkazing.
